This project, by my firends and me is an attempt at reproducing an intrusion detection system, that can detect if a cyber attack has taken place given a record of network logs.        
The network logs can be obtained using CICFlowMeter and passed to the model after standardizing it as a .csv file.  
The model and various components are still being developed and is not yet completed. At present, the model can detect an attack when a series of logs are looped through.  
The dataset used is the CICIDS 2018 dataset that can be found at kaggle - https://www.kaggle.com/datasets/towhidultonmoy/cicids2018
